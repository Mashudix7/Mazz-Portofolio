Apa Saja Dasar HTML?

<h1> - <h6> : Header Tags

Tag ini digunakan untuk membuat heading atau judul dari berbagai tingkat. <h1> adalah yang terbesar dan <h6> adalah yang terkecil.

Contoh penggunaan:

<h1>Heading 1</h1>

<p> : Paragraph

Tag ini digunakan untuk membuat paragraf.

Contoh penggunaan:

<p>Ini adalah sebuah paragraf.</p>

<a> : Anchor

Tag ini digunakan untuk membuat tautan atau link.

Contoh penggunaan:

<a href="https://example.com">Klik disini</a>

<img> : Image

Tag ini digunakan untuk menampilkan gambar.

Contoh penggunaan:

<img src="image.jpg" alt="Deskripsi gambar">

<ul> dan <li> : Unordered List

Tag ini digunakan untuk membuat daftar yang tidak berurutan.

Contoh penggunaan:

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

<ol> dan <li> : Ordered List

Tag ini digunakan untuk membuat daftar yang berurutan.

Contoh penggunaan:

<ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ol>

<div> : Division

Tag ini digunakan untuk membagi konten menjadi bagian-bagian atau divisi.

Contoh penggunaan:

<div>Ini adalah sebuah div.</div>

<span> : Span

Tag ini digunakan untuk membungkus teks atau elemen lainnya untuk tujuan styling.

Contoh penggunaan:

<span class="highlight">Teks ini disorot.</span>

<form> : Form

Tag ini digunakan untuk membuat form input data.

Contoh penggunaan:

<form>
    <input type="text" name="name">
    <input type="submit" value="Submit">
</form>

<table> : Table

Tag ini digunakan untuk membuat tabel.

Contoh penggunaan:

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

<header> : Header

Tag ini digunakan untuk mendefinisikan bagian header dari sebuah dokumen atau section.

Contoh penggunaan:

<header>Ini adalah header</header>

<footer> : Footer

Tag ini digunakan untuk mendefinisikan bagian footer dari sebuah dokumen atau section.

Contoh penggunaan:

<footer>Ini adalah footer</footer>

<section> : Section

Tag ini digunakan untuk mendefinisikan sebuah section dalam dokumen HTML.

Contoh penggunaan:

<section>Ini adalah section</section>

<article> : Article

Tag ini digunakan untuk mendefinisikan konten artikel independen dalam dokumen HTML.

Contoh penggunaan:

<article>Ini adalah artikel</article>

<aside> : Aside

Tag ini digunakan untuk mendefinisikan konten yang tidak berhubungan langsung dengan konten utama.

Contoh penggunaan:

<aside>Ini adalah aside</aside>

<main> : Main

Tag ini digunakan untuk mendefinisikan konten utama dalam dokumen HTML.

Contoh penggunaan:

<main>Ini adalah konten utama</main>

<label> : Label

Tag ini digunakan untuk mendefinisikan label untuk elemen form.

Contoh penggunaan:

<label for="name">Nama:</label>

<input> : Input

Tag ini digunakan untuk membuat elemen input dalam form.

Contoh penggunaan:

<input type="text" name="name">

<button> : Button

Tag ini digunakan untuk membuat tombol dalam form atau dokumen HTML.

Contoh penggunaan:

<button type="button">Klik saya</button>

<iframe> : Iframe

Tag ini digunakan untuk menyematkan dokumen HTML lainnya dalam dokumen HTML saat ini.

Contoh penggunaan:

<iframe src="https://example.com" 
   width="600" height="400"></iframe>
